Answer 1: 

Tocken : On the command line, each sequence of non-blank characters is a token 
 	 or a word.

Argument : An argument is a file name or other data (path of the file) that is 
	   provided to a command in order for a command to use it as an input.

Option : Option is used to change the behaviour of the command.


Argument and Option : Option is used to change the behaviour of command.
		      Argument is the name of the file or path.

		     Example : head -10 file1

	     Here, 'head' is the command, '-10' is the option and 'file1' is the argument.


Answer 4: Utilities which are buit in shell are called as builtins. 
	  A built-in command is a part of shell itself. A utility is an 
	  executable program that is installed (usually in a system directory)
	  which the shell executes. We have used cd, cat, cp, echo builtins.
